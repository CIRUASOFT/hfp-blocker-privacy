# hfp-blocker-privacy

Public hosting for the privacy policy and landing page of the
Android app **Disable HFP (Hands-Free Mode)** (package
`com.cirua.disablehfp`, publisher CIRUA).

The app's source code lives in a separate private repository.
This repo exists only so the privacy policy URL Google Play
Console requires can be served from a free GitHub Pages site
without making the source code public.

## Pages

- [`index.html`](index.html) — landing page.
- [`privacy.html`](privacy.html) — privacy policy. The Play Store
  listing's privacy URL points here.

## Live URLs

- https://pavloskaza.github.io/hfp-blocker-privacy/
- https://pavloskaza.github.io/hfp-blocker-privacy/privacy.html

## Updating

When the privacy or landing copy changes in the private code
repo, copy the two HTML files into this repo and push. The
manual step is intentional — it forces a review of the wording
before it goes live to the public.
